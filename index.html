INDRA

INDRA (Integrated Network and Dynamical Reasoning Assembler) is an automated model assembly system interfacing with NLP systems and databases to collect knowledge, and through a process of assembly, produce causal graphs and dynamical models.
Read the documentation.
See the code on GitHub.
The INDRA REST API.

INDRA DB

The INDRA Database is a framework for creating, maintaining, and accessing a database of content, readings, and statements. You can search the INDRA Database for statements here: db.indra.bio.
Here is an example of searching for statements where ACE2 is object while returning a maximum of 100 results: https://db.indra.bio/statements/from_agents?object=ACE2&max_stmts=100&format=html
The INDRA Database has a REST API, documented here.
Read the documentation here.

The INDRA Network Search

The INDRA Network Search service allows you to do pathfinding in a graph representation of a snapshot of the INDRA database. The service is available at network.indra.bio.

Clare

CLARE (Context-aware Language-based Reasoning Engine) is a dialogue system that can be used to talk with a machine partner about molecular biology and can be adapted for us in other domains. The context awareness allows the user to fine tune results by asking follow up questions while natural language model building allows the user to build a model of e.g. a drug mechanism and ask qualitative questions about the behavior of the model.
See an introductory video about CLARE here.

BOB

Bob with Bioagents is a machine partner you can chat about molecular biology and solve problems together with.
With the help of BOB a user can formulate a hypothesis for e.g. "How does SB525334 decrease the amount of SMURF2?" (a hypothetical experimental observation). The hypothesis is then turned into a mathematical model of biochemical mechanisms that is able to verify that the model reproduces observations of interest. A video showing this example is found here.
Another usage example for BOB is analyzing a list of genes and experimental data uploaded by the user by e.g. performing gene set enrichment analysis on the genes. A video for this example is available here.
BOB is available at dialogue.bio.
