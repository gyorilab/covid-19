INDRA

INDRA (Integrated Network and Dynamical Reasoning Assembler) is an automated model assembly system interfacing with NLP systems and databases to collect knowledge, and through a process of assembly, produce causal graphs and dynamical models. See the documentation and code at GitHub for more details.

INDRA DB

The INDRA Database is a framework for creating, maintaining, and accessing a database of content, readings, and statements. Read the documentation here and the details about the REST API here.

The INDRA Network Search

The INDRA Network Search service allows you to do pathfinding in a graph representation of a snapshot of the INDRA database. The full documentation for the network search UI is available via readthedocs here.

Clare

CLARE (Context-aware Language-based Reasoning Engine) is a dialogue system that can be used to talk with a machine partner about molecular biology and can be adapted for us in other domains. See an introductory video about CLARE here.

BOB

Bob with Bioagents is a machine partner you can chat about molecular biology and solve problems together with. BOB is found at dialogue.bio.
