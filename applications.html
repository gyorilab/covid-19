INDRA: Integrated Network and Dynamical Reasoning Assembler

subtitle subtitle subtitle subtitle

INDRA is an automated model assembly system interfacing with NLP systems and databases to collect knowledge, and through a process of assembly, produce causal graphs and dynamical models.

Knowledge Sources

INDRA utilises multiple natural language processing systems and structured databases

REST API

INDRA has a REST API available at http://api.indra.bio:8000 where a lot of INDRA's functionalities can be tested out. Note that the REST API is ideal for prototyping and for building light-weight web apps, but should not be used for large reading and assembly workflows.
